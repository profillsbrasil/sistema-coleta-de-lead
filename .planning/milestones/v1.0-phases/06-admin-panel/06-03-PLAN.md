   - Items: map from listVendors query, display name + truncated userId
   - On change: set selectedVendor, reset page to 1

c) When vendor selected, show vendor stats row (3 StatCards in a grid):
   - Total de Leads: from query total
   - Score: calculate from leads data (quente*3 + morno*2 + frio*1) OR use a separate query
   - Leads Hoje: filter leads by createdAt >= today
   Actually, to keep it simple and avoid extra queries, use the `total` from listByUser response. For score and today's count, do a client-side calculation from the returned leads data, or add these to the listByUser response in the router. Better approach: the admin stats router already has `getGlobalStats` with userId filter — use `trpc.admin.stats.getGlobalStats.useQuery({ userId: selectedVendor }, { enabled: !!selectedVendor })` for stats.

d) Pagination info: `<p className="text-muted-foreground text-sm text-right mb-2">Mostrando {start}-{end} de {total}</p>` where start = offset+1, end = min(offset+limit, total).

e) Table with columns per UI-SPEC:

   - Telefone/Email: `lead.phone || lead.email || "-"`

i) Empty states per UI-SPEC:

Map fields: ensure `localId`, `name`, `phone`, `email`, `company`, `position`, `segment`, `notes`, `interestTag` are present. Set `photo: null` (admin doesn't manage photos locally).
- Create `onUpdate` callback that calls `trpc.admin.leads.update.useMutation`:
```typescript
const updateMutation = trpc.admin.leads.update.useMutation({
  onSuccess: () => {
    toast.success("Lead atualizado!");
    router.push("/admin/leads" as unknown as "/");
  },
  onError: () => {
    toast.error("Erro ao atualizar lead.");
  },
});

async function handleUpdate(data: LeadFormData) {
  await updateMutation.mutateAsync({
    localId: leadId,
    data: {
      name: data.name,
      phone: data.phone ?? null,
      email: data.email ?? null,
      company: data.company ?? null,
      position: data.position ?? null,
      segment: data.segment ?? null,
      notes: data.notes ?? null,
      interestTag: data.interestTag,
    },
  });
}
```

- Create `onDelete` callback using `trpc.admin.leads.delete.useMutation`:
```typescript
async function handleDelete() {
  await deleteMutation.mutateAsync({ localId: leadId });
  toast.success("Lead excluido!");
  router.push("/admin/leads" as unknown as "/");
}
```

```tsx
<LeadForm
```
